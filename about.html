About

Hi, I'm Alfonso Espino. I'm 13 and I'm into coding and engineering stuff. I actually learned about the monty hall problem relatively recently but it really interested me. This is because almost anyone (including me) would intuitively, almost automatically assume that it really doesn't make a difference whether you switch or stay, however if you use logic you can realize that you actually have more information than you thought. It reminds me how we should think logically about things, instead of making intuitive assumptions. I've been wanting to make a project on the monty hall problem for a while, but I finally decided to create it, and also hackclub has a YSWS (You Ship We ship) where you can earn ice-cream for creating a website, so I figured why not make this for my submission. It took me a while to understand it so to ensure that I understand it, I figure the best way is by trying to explain it to others. Originally I was just going to make this a simulation but I decided that I should have multiple pages to provide a better site with my (not very good) explanation, a simulation, a home page, and lastly the about page you are reading right now.
